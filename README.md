# 今日去邊度行山？⛰️
## Hong Kong Hiking Trail Randomizer

*Made by 港大行社 The Ramblers' Club, HKU ⛰️| [@hku_ramblers](https://www.instagram.com/hku_ramblers)*

---

## Overview

A single-file HTML web application that randomly selects a Hong Kong hiking trail based on user-defined filters. Designed for members of The Ramblers' Club, HKU (港大行社) and any hiking enthusiasts who want a fun, effortless way to decide where to hike.

The app contains a database of 400+ hiking routes across Hong Kong, sourced from [@oasistrek](https://www.instagram.com/oasistrek). Each trail includes five data points: name, scenery rating, difficulty, distance, and estimated duration.

---

## Features

- **Scenery filter** — select which scenery ratings (0.5–5 stars) to include
- **Difficulty range slider** — set a minimum and maximum difficulty level
- **Distance range slider** — filter by trail length in kilometres
- **Duration range slider** — filter by estimated hiking time in hours
- **Live trail count** — shows how many trails match your current filters
- **Random selection** — press the main button to draw a random trail from the filtered pool
- **Result card** — displays trail name, scenery stars, difficulty bar, distance, and duration
- **Re-roll** — draw another trail without resetting your filters
- **Google Search** — opens a Google search for the selected trail in a new tab

---

## How to Use

### Step 1 — Set your filters
Use the filter panel to narrow down trails by your preferences:
- Tap the scenery rating chips to toggle which scenery scores to include or exclude
- Drag the **Difficulty** sliders to set your desired range (e.g. 3–4.5)
- Drag the **Distance** sliders to set a km range (e.g. 5–15 km)
- Drag the **Duration** sliders to set a time range (e.g. 2–5 hours)

### Step 2 — Roll
Press the **今日去邊度行山** button. A trail will be randomly selected from all routes matching your filters.

### Step 3 — Go hiking!
The result card shows the trail details. You can re-roll for another suggestion, or tap the search button to find more information about the trail online.

## Trail Data Fields

| Field | Description |
|---|---|
| Name (名稱) | Chinese name of the hiking trail or area |
| Scenery (景觀) | Scenic rating from 0.5 to 5 stars |
| Difficulty (難度) | Difficulty rating from 0.5 (very easy) to 5 (extreme) |
| Distance (長度) | Total trail length in kilometres |
| Duration (需時) | Estimated time to complete the trail in hours |
